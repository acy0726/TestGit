1. 서  론
  전 세계적으로 스마트 기기와 PC의 보급률 증가로 웹 사이트 사용자들이 급격하게 증가하고 있다. 웹 사이트 사용자들은 개인 취향에 맞춘 검색, 뉴스, 상품 추천 등 다양한 개인화 서비스를 제공받기 원한다. 웹 사이트 운영 기업들은 이러한 요구사항을 해결하기 위해 사용자들에게 회원가입을 하게하여 개인화 서비스를 제공하고 있다. 현재 이러한 회원가입 방식의 개인화 서비스는 회원가입을 원하지 않는 사용자에게 개인화 서비스를 할 수 없다는 문제점이 있다. 현재 수많은 웹 사이트가 생성이 되고 그에 따라 사용자들은 다양한 웹 사이트를 이용하지만 각각의 사이트마다 회원가입을 해야 한다는 문제점이 있고 기업들이 관리하고 있는 개인정보에 대한 유출사고와 원하지 않는 범위에서 개인정보가 사용되는 거부감으로 인해[1] 회원가입을 원하지 않는 웹 사이트 사용자들이 늘어나고 있다. 이러한 문제점들을 해결하기 위해서는 웹 사이트의 회원가입 없이 비로그인 사용자에게 개인화 서비스를 제공 할 수 있는 방법이 필요하다.

비로그인 웹 사이트 사용자에게 무의식적 개인화 서비스를 제공하기 위해서는 사용자 측면에서 실행되는 개인화 서비스 아키텍처가 구현되어야 한다. 
사용자 측면에서 개인화 서비스가 실행되면 회원가입의 필요성을 제거 할 수 있으며 기업에게 개인정보 제공을 최소화 할 수 있는 이점이 발생된다. 사용자 측면의 개인화 서비스 구현을 위해서는 사용자가 웹 사이트를 접속 하였을 때 실행되는 객체기반 클라이언트 스크립트 언어 JAVASCRIPT(JS) 기반 구현이 필요하다. JS는 웹 브라우저 내에 다양한 구현 기능을 제공하며 많은 수의 검증된 라이브러리를 지원한다. 그리고 사용자의 웹 사이트 이용 패턴을 학습하기 위해서는 클라이언트 기반 딥러닝(deeplearning) 구현이 필요하다.

본 논문에서는 JS기반 딥러닝 라이브러리 Tensorflow.js(TFJS)를 사용하여 검색 카테고리 추천 시스템을 구현하여 웹 사이트 개인화 방안을 제시한다. 비로그인 사용자의 검색 카테고리 선택 정보를 검색 창에 미리 제시함으로써 개인화 서비스를 제공한다. W3C가 권고한 웹 브라우저 표준 데이터베이스 Indexed Database를 이용하여 사용자의 검색 카테고리 선택 정보와 검색한 시간정보를 저장하고 저장된 정보를 TFJS로 구현한 신경망을 통해 웹 사이트에 접속한 시간과 과거의 비슷한 시간대의 검색 카테고리 선택 정보를 학습하여 사용자가 높은 확률로 선택하게 될 검색 카테고리 정보를 검색 창에 제공한다.

본 논문의 구성은 2장에 관련연구를 설명하고 3장에서는 검색 카테고리 추천 시스템을 설명한다. 4장에서는 실험 결과를 보여주며 5장에서는 결론을 나타낸다.
